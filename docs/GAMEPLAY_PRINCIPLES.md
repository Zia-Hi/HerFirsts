# Gameplay Principles

Every interaction teaches a real-life skill.

Every failure is reversible.

Players should never be punished for curiosity.

Safety comes before efficiency.

Every mission ends with a knowledge card. Knowledge Cards are permanent.

Players should always understand WHY they are doing something. Not just HOW.

Tutorials should be environmental.

The player should rarely read long text.

Interaction should feel tactile. Objects have weight.Tools matter. Actions matter.

The game should reduce anxiety, not create anxiety.

Every completed mission should increase confidence.

The player is growing. Not just solving problems. The apartment changes. The character changes. The player's confidence changes.
