# Project Summary

Her Firsts is a cozy first-person narrative game that teaches real independent living skills through interactive gameplay.

It is NOT a website.

It is NOT an educational slideshow.

It is NOT a simulator.

The player learns naturally by interacting with the world.

Every mission represents one "first" in adulthood.

The purpose is to reduce anxiety when facing unfamiliar real-life situations.

Players should finish each mission thinking:

"I think I could actually do this in real life."

---

# Core Vision

Confidence comes from experience.

The game is about becoming independent, one small victory at a time.

The educational content is hidden inside meaningful gameplay.

Learning should never interrupt immersion.
