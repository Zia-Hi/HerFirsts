# Art Direction

## Visual Identity

This game should feel like an animated film rather than a website.

Every screen should be beautiful enough to become wallpaper.

Glassmorphism

Neon

Bright RGB

Startup UI

Floating Cards

Hero Banner

Over-designed Menus

Rounded SaaS Cards

Dashboard Layouts

Large gradients

Oversized shadows

Modern app aesthetics

---

## UI

Everything should feel like it belongs to the world.

No floating UI.

No modern website components.

No rounded SaaS cards.

No gradients unless they imitate sunlight.

Examples:

Door instead of Start Button.

Notebook instead of Menu.

Toolbox instead of Inventory Window.

---

# Composition

Avoid centered layouts.

Avoid website hero sections.

Every composition should tell a story.

Objects should guide attention naturally.

---

## Camera

First Person

Slow

Stable

Human

Never exaggerated.

---

## Feeling

The player should feel:

"I've finally moved into my own home."
